# Vue Loan Calculator
The project took the [PHP Loan Calculator](https://github.com/snake14/php-loan-calculator) project and converted it from PHP to Vue.js so that it is strictly client-side and doesn't need a PHP server.

To use this project, download it somewhere and simply open the index.html file in a browser. You can also drop it into a directory made public via Apache or similar web server. I'm using Github Pages.
